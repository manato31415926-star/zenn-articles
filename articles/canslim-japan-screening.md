# 日本株でCAN SLIMスクリーニングを実践するコツ

なぜこれが重要か
--------------------------------

投資初心者にとって、株の値上がりや下落の予測にはさまざまな方法があります。しかし、それらの方法の多くは、長い時間をかけて株価を分析することが求められます。これが難しい理由の1つは、分析に必要な情報量が多いことです。たとえば、株の時価総額が100億円で、その年間売上高が10倍以上あがるなど、株の成長可能性を判断するのに時間や経験が必要になります。

また、投資初心者は、株価の変動に影響を与える要因をすべて考慮することが難しくて、予測できません。たとえば、経済的環境、市場情勢、経営陣などの要因があります。これらの要因について調査すると時間がかかります。

これらの問題を解決する方法の1つがCAN SLIM投資法です。この投資法は、株価の上昇に影響を与える要因を簡潔にまとめており、その要因を分析して投資を行うことができます。したがって、この記事ではCAN SLIM投資法の基本と日本株での実践方法について解説します。

**CAN SLIM投資法の基本**
-----------------------------

CAN SLIMは、1970年代にWiley Investment Conferenceで初めて発表された投資手法です。この投資手法は、企業の財務状況や市場環境を分析することから始まりますが、その後、株価の上昇に影響を与える要因を簡潔にまとめました。

CAN SLIM投資法では、株を選ぶために6つの要因を考慮します。それらは次の通りです。

*   **C** : 大きな成長
*   **A** : 稼働率が高い
*   **N** : 株価の伸びが速い
*   **S** : 株式公開後一定期間、株価の上昇が続いている
*   **L** : 株式の流動性が高く、取引が容易であること
*   **I** : 株式のイニシアチブ力が高い

これらの要因については、次の順序で分析します。

### 1. 大きな成長(C)

大きな成長というのは、企業が過去5年間で平均20%以上の売上高を伸ばしたことです。企業が大きな成長を見せる場合、その企業株価も大きく上昇する可能性があります。

たとえば、2022年にTSMC（台湾セミコンダクターモールズ）が発表した財務データによると、過去5年間で売上高が平均30%以上伸ばされました。同社の株価も、2020年の時点から2022年までに約6倍上昇しました。

### 2. 稼働率が高い(A)
