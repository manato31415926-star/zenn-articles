# 日経平均の「5万5000円」：通過点か天井か　CAN SLIM投資を適用して見る

## CAN SLIM投資とは何か？

CAN SLIM投資は、ウィリアム・オニールが開発した株式投資手法です。株価上昇に伴う資金調達の増加、企業の成長性、技術革新、高い収益性、市場リーダーシップを重視するものです。日本株の成長株投資においても、この手法を適用することで、企業の潜在力と将来性をより深く理解し、投資に成功することができます。

### CAN SLIMの5つの要素

1.  **キャプチャリング（C）**：企業が市場リーダーシップを握る能力。経済的状況や技術革新によって起こり得る変化を予測して対処できる企業は、将来性に富んだものと言える。
2.  **アクセラレーター（A）**：資金調達の増加により企業が成長する能力。企業が市場でその力を見せ、その利益を拡大させる能力を重視します。
3.  **ノウハウ（N）**：企業が持つ技術革新と知的財産権。革新的で独自の技術を持つ企業は、競争力を維持し成長を続けることができる。
4.  **スキャンダル（S）**：企業の収益性。企業が一定期間にわたって高収益性を示す能力を重視します。収益性が低い企業は成長する可能性が限られます。
5.  **リーダーシップ（L）**：企業が市場リーダーシップを維持する能力。この点も重要な要素として考慮する必要があります。
