# オニール流「8%損切りルール」の正しい実践方法

なぜオニール流「8%損切りルール」が重要か
=====================================================

投資家にとって、勝ち残るための最も重要なスキルはリスク管理です。株価の波動や市場の変動に直面しても、冷静で厳しい姿勢を維持できることが鍵です。その中でも、オニール流「8%損切りルール」は投資初心者にも有名な概念です。このルールは、株価が8%下落した場合には損切りして、その資金を他の投資機会に回すことを意味しています。

このルールは、CAN SLIM手法の基本原則の一つとして位置づけられています。CAN SLIM手法は、企業の成長性と市場の動向を分析することで株価の上昇傾向を見極める方法です。このルールを理解し、実践することで投資家はリスクを減らし、収益を確保することができます。

**CAN SLIM手法の基本**

CAN SLIM手法には5つの原則があります。

1.  **C - CURRENT QUARTER EARNINGS GROWTH**: 最近の利益成長率が高い企業
2.  **A - ACCOMPLISHMENT IN THE PRIMARY TECKNIQUE** : 主要な技術を確立している企業
3.  **N - NEW PRODUCTS, SERVICES, OR MARKETING**: 新しい製品やサービスを開発中の企業
4.  **S - SUPERIOR MANAGEMENT**: 強力な管理体制を持つ企業
5.  **L - LUCRATIVE INTERNAL ADEQUACY**: 自社で必要な資金を確保している企業

**オニール流「8%損切りルール」の実践方法**

オニール流「8%損切りルール」を実践するには、以下のステップに従うことができます。

1.  **株価の目標**: 株価が目標価格を上回った場合に、買い増しを行う
2.  **8%損切ライン**: 株価が8%下落した場合に、損切りして資金を他の投資機会に回す

**まとめ**

オニール流「8%損切りルール」はリスク管理の重要なスキルです。CAN SLIM手法の基本原則の一つとして位置づけられており、投資初心者にも理解しやすい概念です。このルールを実践することで、投資家は収益を確保しながらリスクを減らし、株価の上昇傾向を見極えることができます。日本株でもこのルールを実践できるので、ご活用ください。
