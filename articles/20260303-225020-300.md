#CAN SLIM投資の資金管理術（1）：日本株での実践方法と具体的な数字

## CAN SLIM投資の基本

ウィリアム・オニール氏が提唱したCAN SLIM投資は、資金を適切に管理することで投資結果を高めるための投資手法です。資金管理は投資における重要な要素であり、どの投資方法でも同じですが、キャパシティ（C）、アドバンテージ（A）、ニュートラリティ（N）、スピード（S）、イマネンス（I）、モメンタム（M）という6つの要素を組み合わせることで利益を生むことができます。

### キャパシティ（C）

キャパシティは、投資のリスクとリターンのバランスを保つことです。リスクは高くなければならないですが、同時に高いリターンも期待できなければなりません。一般的には、キャパシティが80%以上の場合、リスクが過大になります。

## 資金管理の重要性

資金管理は投資において非常に重要な要素です。適切な資金管理を行うことで、投資のリスクを減らしリターンを高めることができます。ここでは、日本株での実践方法と具体的な数字について説明します。

### 資金の分割

資金の分割は、リスクを分散させるために重要です。一般的には、資金を50%程度投資し、残りの半分は保有することになります。

| 分割額 | 利益率 |
| --- | --- |
| 40%以下 | 中下程度（平均5-7％） |
| 40-60% | 高い程度（平均10-15％） |

### 資金の増加

資金の増加は、投資で得た利益を追加することで行うことができます。まず、30%以上の利益が得られた場合、次の週に50%程度の額を追加します。

| 増加額 | 利益率 |
| --- | --- |
| 10-20% | 中下程度（平均5-7％） |
| 30-40% | 高い程度（平均10-15％） |

## 日本株での実践方法

ここでは、日本株でのCAN SLIM投資の実践方法について説明します。

### 資金の選択

日本株で資金を選ぶ際は、キャパシティ、リスク、リターンのバランスを考慮する必要があります。一般的には、以下のような条件を満たす会社を選定します。

| 条件 | 値 |
| --- | --- |
| 市況価格 | 5000円以下 |
| EPS | 100円以上 |
| PBR | 2.5倍以下 |

### 資金の増加

資金の増加は、日本株でも同様に実施できます。まず、30%以上の利益が得られた場合、次の週に50%程度の額を追加します。
